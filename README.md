# starforce-js
Simulates the starforce function in the game MapleStory.

Hosted on github.io [here](https://doovid96.github.io.starforce.js/).

Inputs:
* Equipment Level
* Initial Starforce
* Goal Starforce
* Starcatch (0 to 21)
* Safeguard (12 to 16)
* Sunny Sunday
* MVP Discount

Outputs:
* Mesos
* Destroyed Equips
* Chance Times
* Enhancements
* Sequence of starforce levels in simulation
